# Punk-Records-v1-Menggunakan-FastAPI-untuk-Keamanan-dan-Integritas-Data
Implementasi Layanan Kriptografi pada Punk Records-v1 Menggunakan FastAPI untuk Keamanan dan Integritas Data

Repository ini berisi implementasi layanan kriptografi terintegrasi pada sistem Punk Records-v1 yang dikembangkan menggunakan FastAPI. Proyek ini bertujuan untuk meningkatkan keamanan komunikasi data antar pengguna dengan menjamin autentikasi, integritas, dan kerahasiaan data pada layanan berbasis API.

Sistem ini mengimplementasikan digital signature berbasis ECDSA untuk verifikasi keaslian pesan dan dokumen, serta hybrid cryptography yang mengombinasikan Elliptic Curve Diffieâ€“Hellman (ECDH) untuk pertukaran kunci aman dan AES-GCM untuk enkripsi data. Selain itu, layanan ini dilengkapi dengan manajemen secure session, penyimpanan public key, mekanisme relay pesan terenkripsi, serta penandatanganan dokumen PDF.

Implementasi ini diharapkan dapat menjadi referensi penerapan kriptografi modern pada layanan API berbasis Python, khususnya dalam membangun sistem komunikasi digital yang aman dan terpercaya.
